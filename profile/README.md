# 开源观察｜FOSScope

开源观察（FOSScope）是一个开源相关的资讯站，专注于开源技术相关的新闻、教程、工具、资源等内容，旨在更好的推广开源技术，让更多的人了解和使用开源技术。

我们的主要工作内容包括：

- 收集开源相关新闻并撰写相关文章
- 撰写开源技术相关的教程
- 翻译国外优秀的开源技术文章

## 如何加入我们

目前 FOSScope 仍在初创阶段，我们目前非常欢迎有兴趣参与各项工作的小伙伴加入我们，具体加入方式如下：

1. 访问 [加入申请 issue](https://github.com/FOSScope/.github/issues/1)
2. 在 issue 中留下你的 GitHub ID 并说明你希望加入 FOSScope
3. 我们会尽快邀请你加入 GitHub 组织
